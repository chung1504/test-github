**Hiểu nhanh**

- git fetch : lấy code mới về nhưng khoong merge vào nhánh hiện tại
- git fetch origin main

- git pull : lấy code mới về và merge (gộp) vào code luôn
- git pull origin main

- git push : Dấy code từ máy (local) lên remote
- git push -u origin main (lần đầu đẩy lên)
- git push ( những lần sau chỉ cần vậy )
